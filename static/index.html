DiscordSafeEmbed API

Create an Embed

GET /manual

Query string parameters:

  • title: The title of the embed. Must be less than 256 characters.
  • description: The string that goes into the body of the embed. Must be less than 2048 characters.
  • image: A link to an image.
  • siteName: The small text that goes above the title. Must be less than 256 characters.
  • color: The color of the embed, in rrggbb format (without the #). Must be exactly 6 characters long if used.

Embedding the manual link in Discord will create an embed with the given parameters.

POST /api/v1/createEmbed

JSON request body parameters:

  • providerName: The provider name field of the embed. Must be less than 256 characters.
  • providerUrl: The link associated with the provider name.
  • authorName: The author name field of the embed. Must be less than 256 characters.
  • authorUrl: The link associated with the author name.
  • title: The title of the embed. Must be less than 256 characters.
  • description: The string that goes into the body of the embed. Must be less than 2048 characters.
  • image: A link to an image.
  • type: The type of embed.
  • color: The color of the embed, in rrggbb format (without the #). Must be exactly 6 characters long if used.
  • siteName (DEPRECATED): Use providerName instead. The small text that goes above the title. Must be less than 256 characters.

Rate Limit: 10 per minute

Returns a string (embed ID). The embed ID will only be valid for 5 minutes. Once sent in Discord, the embed will not change, but editing the message will delete the embed!

/e/:id

Route parameters:

  • :id: The ID of the embed.

Send this link into Discord with an embed ID to display the embed.
